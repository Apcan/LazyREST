#LazyREST-v2

> LazyREST-v2 整合了SpingBoot以及SpingDataJpa等Sping全家桶的脚手架，方便快速产出RESTful接口

#LazyREST-v1

> LazyREST的v1版本请查看v1分支,基础版本，基于SSM

## 技术选型
* SpringBoot 1.5.8
* SpingDataJpa
* Shiro 1.2.2
* Druid 1.0.19
* JDK 1.8
* Maven

## 技术实现

* 实现基于RESTful风格规范的接口
* 实现正常业务日志及异常日志的记录
* 实现接口调用访问的安全性校验
